Excel Formula Analysis Project

# Project Overview
This project focuses on analyzing employee payroll and sales performance data using advanced Excel formulas and pivot tables.

# Datasets Used
- Employee Payroll Data
- Sales Performance Data

# Excel Features & Formulas Used
- VLOOKUP
- XLOOKUP
- HLOOKUP
- INDEX and MATCH
- IF conditions
- COUNTIF
- MIN and MAX
- FILTER function
- Pivot Tables (multiple types)

# Key Learnings
- Data lookup and matching techniques
- Conditional data analysis
- Summarizing data using pivot tables
- Improving accuracy and efficiency in Excel analysis

# Files Included
- employee_payroll_analysis.xlsx
- sales_performance_analysis.xlsx
